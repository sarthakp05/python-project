# Hospital Management System 🏥

A robust, production-ready Hospital Management System built with Python, Flask, SQLAlchemy, and Bootstrap 5. This application provides a comprehensive solution for managing daily hospital operations, streamlining workflows for administrators, doctors, and receptionists.

## ✨ Features

*   **Role-Based Access Control (RBAC):** Secure login and customized dashboards for different staff roles:
    *   **Admin:** Full system control, user management, and high-level analytics.
    *   **Doctor:** Manage appointments, view patient medical history, and write prescriptions.
    *   **Receptionist:** Handle patient registration, schedule appointments, and manage billing.
*   **Patient Management:** Comprehensive patient life-cycle tracking and electronic medical records (EMR).
*   **Appointment Scheduling:** Interactive calendar system for booking and managing doctor schedules.
*   **Prescription Management:** Digital prescription generation and tracking.
*   **Billing & Invoicing:** Streamlined billing system for patient treatments and consultations.
*   **Interactive Dashboards:** Data visualization and analytics using integrated charts.
*   **Responsive Design:** Mobile-friendly UI built with Bootstrap 5.

## 🛠️ Technology Stack

*   **Backend:** Python 3, Flask
*   **Database:** SQLite / PostgreSQL (via SQLAlchemy ORM)
*   **Frontend:** HTML5, CSS3, Bootstrap 5, Jinja2 Templates
*   **Forms:** Flask-WTF
*   **Authentication:** Flask-Login
